## 📂 Array Programs

| # | Program | Description |
|---|---------|-------------|
| 1 | Frequency | Count the frequency of each element in an array |
| 2 | LargestNumber | Find the largest element in an array |
| 3 | MaximumProductSubarray | Find the contiguous subarray with the maximum product |
| 4 | NonRepeating | Find the first non-repeating element in an array |
| 5 | OddEven | Count or separate odd and even elements |
| 6 | RemoveDuplicates | Remove duplicate elements from an array |
| 7 | RepeatingElements | Find all repeating elements in an array |
| 8 | Reverse | Reverse the elements of an array |
| 9 | SecondSmallest | Find the second smallest element |
| 10 | SmallestNumber | Find the smallest element in an array |
| 11 | Sort | Sort the array in ascending order |
| 12 | Sum | Calculate the sum of all array elements |
| 13 | SymmetricPairs | Find symmetric pairs in a collection of pairs |
| 14 | TripletsSum | Find triplets whose sum equals a target value |
| 15 | TwoArraysEqual | Check whether two arrays are equal |

## 📂 String Programs

| # | Program | Description |

## 🛠️ How to Run

### Arrays

```bash
cd Arrays
javac FileName.java
java FileName
```

### Strings

```bash
cd Strings
javac FileName.java
java FileName
```

## 📚 Topics Covered

### Arrays
- Array Traversal
- Searching (Min, Max, Second Smallest)
- Sorting
- Hashing (HashMap, HashSet)
- Subarray Problems
- Two-pointer / Brute Force Approaches

### Strings
- String Manipulation
- Character Frequency
- Palindrome Checking
- Anagram Detection
- Case Conversion
- Lexicographical Comparison
- String Sorting
- Pattern Processing
