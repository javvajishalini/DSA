# DSA - Data Structures and Algorithms

A collection of Java programs for practicing Data Structures and Algorithms.

## 📁 Structure

```
DSA/
└── Arrays/
    ├── Frequency.java
    ├── LargestNumber.java
    ├── MaximumProductSubarray.java
    ├── NonRepeating.java
    ├── OddEven.java
    ├── RemoveDuplicates.java
    ├── RepeatingElements.java
    ├── Reverse.java
    ├── SecondSmallest.java
    ├── SmallestNumber.java
    ├── Sort.java
    ├── Sum.java
    ├── SymmetricPairs.java
    ├── TripletsSum.java
    └── TwoArraysEqual.java
```

## 📌 Array Programs

| # | Program | Description |
|---|---------|-------------|
| 1 | Frequency | Find frequency of each element in an array using HashMap |
| 2 | LargestNumber | Find the largest/maximum element in an array |
| 3 | MaximumProductSubarray | Find the maximum product subarray (Kadane's variant) |
| 4 | NonRepeating | Find non-repeating elements in an array |
| 5 | OddEven | Count odd and even numbers in an array |
| 6 | RemoveDuplicates | Remove duplicate elements using HashSet |
| 7 | RepeatingElements | Find repeating/duplicate elements in an array |
| 8 | Reverse | Reverse an array using Collections |
| 9 | SecondSmallest | Find the second smallest element in an array |
| 10 | SmallestNumber | Find the smallest/minimum element in an array |
| 11 | Sort | Sort an array using Collections.sort |
| 12 | Sum | Calculate sum of all elements in an array |
| 13 | SymmetricPairs | Find symmetric pairs in a 2D array |
| 14 | TripletsSum | Find triplets that sum to a target value |
| 15 | TwoArraysEqual | Check if two arrays are equal |

## 🛠️ How to Run

```bash
cd Arrays
javac <FileName>.java
java <ClassName>
```

## 📚 Topics Covered
- Array Traversal
- Searching (Min, Max, Second Smallest)
- Sorting
- Hashing (HashMap, HashSet)
- Subarray Problems
- Two-pointer / Brute Force approaches
